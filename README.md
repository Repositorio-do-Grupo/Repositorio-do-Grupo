# Segurança para Negócios

`CURSO: CST em Segurança da Informação`

`SEMESTRE: Eixo 2`

O foco deste projeto no Eixo 2 consiste em desenvolver estratégias de Segurança para Negócios, integrando tecnologias de defesa à viabilidade operacional das empresas. Sob a orientação do professor Will Machado, o trabalho prioriza a proteção de ativos críticos e a mitigação de riscos, garantindo que a infraestrutura tecnológica suporte a continuidade e a integridade dos processos organizacionais.
